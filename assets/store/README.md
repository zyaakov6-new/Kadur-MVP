# App Store Assets

This directory contains the required assets for App Store and Google Play Store submission.

## Required Screenshots

### iOS
- **5.5" Display (iPhone 8 Plus):** 1242 x 2208 px
- **6.5" Display (iPhone 11 Pro Max):** 1242 x 2688 px

### Android
- **Feature Graphic:** 1024 x 500 px
- **Phone Screenshots:** Minimum 320px, Max 3840px (16:9 aspect ratio recommended)

## Instructions
Replace the placeholder images in these folders with actual screenshots from the app running on the respective simulators/devices.
